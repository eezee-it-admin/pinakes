Microsoft Teams Integration with Odoo Event

Eezee (BeOpen-IT)

Features
  • Each Odoo user can authenticate using their own Microsoft Account
  • Create Microsoft Teams Meetings from Odoo Event
  • Send a meeting invitation email to event attendees with the MS Teams meeting URL

Configuration

    Set "Application ID" and "Client Secret" and then click on "Authenticate" button
    Click here to get Application ID and generate Secret Key

    Login with your Microsoft Account


    After successful authentication, you will see bellow screen. Close the window after authentication.

    The Responsible of the Event must click the button "Create MS Teams Meeting".
    Responsible user must first follow the MS Teams authentication process as shown in above steps.

    Once the meeting is created, email the meeting URL to all the attendees of the event.
